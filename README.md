# ansible-role-system

Ansible role for miscellaneous system configurations that don't fit cleanly in
any of the other roles


## Requirements

    None


## Dependencies

    None


## Role Variables

    None


## Example playbook

    ```yaml
    ---
    - hosts: all

      roles:
          - foolean/system
    ```


## Compliance

    * CIS RedHat Enterprise Linux 8 Benchmark v2.0.0
